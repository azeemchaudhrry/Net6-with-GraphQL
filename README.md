# Net6-with-GraphQL
A demo application to show how to develop a GraphQL API using .Net6, EntityFramework and HotChocolate.

## Tech Stack:
- .Net6
- HotChocolate for GraphQL
- EntityFrameworkCore for ORM
- EntityFrameworkCore.InMemory
